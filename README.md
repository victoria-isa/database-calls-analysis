# Análise de dados de chamados – Python

Este projeto analisa um conjunto de chamados simulados utilizando Python,
Pandas e Matplotlib.

# Funcionalidades
- Estatísticas básicas
- Contagem de chamados por status
- Geração de gráfico de barras
- Visualização rápida do estado do suporte

# Tecnologias
- Python 3
- Pandas
- Matplotlib
